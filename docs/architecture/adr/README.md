# ADR Journal

ADR (Architecture Decision Record) — короткий журнал архітектурних рішень з відповіддю на питання "чому так".

## Статуси

- Proposed
- Accepted
- Superseded
- Deprecated

## Формат імен

- `NNNN-short-title.md`
- приклад: `0001-start-adr-journal.md`

## Як писати ADR

1. Контекст: що боліло або що треба вирішити.
2. Рішення: що саме обираємо.
3. Наслідки: trade-offs, ризики, вплив.
4. Rollback/альтернатива: як відкотити або чим замінити.
