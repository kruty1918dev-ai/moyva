# Модульна документація Moyva

Цей документ є єдиною точкою входу в модульну документацію проєкту.

## Принципи

- Документація згрупована за зонами відповідальності.
- Кожен модуль має власну папку та уніфікований набір піддокументів.
- Логіка описується в одному canonical-модулі без дублювання.
- Для великих блоків використовується згортання/розгортання через details.
