# Voronoi Regions

← [Генератори](../nodes-generators.md) · [Graph System](../README.md)

**Категорія:** Generators · **Файл:** `VoronoiRegionsNode.cs`

Генерує **діаграму Вороного** — розбиває карту на регіони, кожен з яких "належить" найближчій зерновій точці. Корисний для створення провінцій, територій або як базова структура для біомів.

---

## Порти

| Напрямок | Назва | Тип | Опис |
|---|---|---|---|
| **Input** ⚪ | MapWidth | `int` | Ширина карти |
| **Input** ⚪ | MapHeight | `int` | Висота карти |
| **Output** 🟠 | RegionIds | `int[,]` | ID регіону для кожної клітинки (0, 1, 2, ...) |
| **Output** 🟢 | DistanceMap | `float[,]` | Відстань до найближчої зернової точки (0.0–1.0) |

## Параметри

| Параметр | Тип | За замовч. | Опис |
|---|---|---|---|
| **Region Count** | `int` | 12 | Кількість регіонів (2–100) |
| **Seed** | `int` | 0 | Зерно для позицій точок |
| **Distance Map** | `bool` | false | Чи обчислювати карту відстаней |

## Ігрове застосування

- **Провінції на карті** — кожен регіон = окрема територія
- **Різні біоми** — комбінуйте з `ConditionalSwitch` для призначення різних тайлів кожному регіону
- **DistanceMap** — використовуйте як маску для плавних переходів на кордонах
