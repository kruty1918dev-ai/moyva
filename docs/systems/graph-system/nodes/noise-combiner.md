# Noise Combiner

← [Обробка](../nodes-processing.md) · [Graph System](../README.md)

**Категорія:** Processing · **Файл:** `NoiseCombinerNode.cs`

Комбінує **дві карти висот** в одну за допомогою математичної операції. Ідеально для створення складного рельєфу з простих компонентів.

---

## Порти

| Напрямок | Назва | Тип | Опис |
|---|---|---|---|
| **Input** 🟢 | HeightMap A | `float[,]` | Перша карта висот |
| **Input** 🟢 | HeightMap B | `float[,]` | Друга карта висот |
| **Output** 🟢 | Combined | `float[,]` | Результат комбінування |

## Параметри

| Параметр | Тип | За замовч. | Опис |
|---|---|---|---|
| **Mode** | `CombineMode` | Add | Режим комбінування |
| **Blend** | `float` | 0.5 | Вага змішування (тільки для режиму Lerp: 0.0 = повністю A, 1.0 = повністю B) |

## Режими

| Режим | Формула | Коли використовувати |
|---|---|---|
| **Add** | `Clamp01(A + B)` | Додати висоти з двох джерел (з обмеженням 0–1) |
| **Multiply** | `A × B` | Маскування — "прибрати" висоти де B низьке |
| **Max** | `max(A, B)` | Залишити вище з двох |
| **Min** | `min(A, B)` | Залишити нижче з двох |
| **Lerp** | `lerp(A, B, blend)` | Плавне змішування двох карт |

> **Примітка:** Якщо карти мають різний розмір, менша карта доповнюється нулями.

## Приклад

```
HeightSource(Scale:30)  ──┐
                          ├──► NoiseCombiner(Lerp, 0.3) ──► HeightToTile
HeightSource(Scale:100) ──┘

Результат: Великі пагорби (Scale:100) з дрібними деталями (Scale:30, вага 0.3)
```
