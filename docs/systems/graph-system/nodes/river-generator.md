# River Generator

← [Особливості](../nodes-features.md) · [Graph System](../README.md)

**Категорія:** Features · **Файл:** `RiverNode.cs`

Генерує **річки** від випадкових точок на краю мапи до найнижчої точки карти через A* pathfinding.

---

## Порти

| Напрямок | Назва | Тип | Опис |
|---|---|---|---|
| **Input** 🔵 | BiomeMap | `string[,]` | Базова карта біомів |
| **Input** 🟢 | HeightMap | `float[,]` | Карта висот (визначає шлях річки) |
| **Output** 🔵 | BiomeMap | `string[,]` | Карта з тайлами річок |
| **Output** 🔵 | ObjectMap | `string[,]` | Карта з об'єктами вздовж річки (мости тощо) |

## Параметри

| Параметр | Тип | Опис |
|---|---|---|
| **Base Object Id** | `string` | ID об'єкта річки в ObjectMap |
| **Rivers Count** | `int` | Кількість річок для генерації |

## Як працює

1. Збираються всі точки на краях карти й перемішуються випадково
2. Знаходиться найнижча точка на карті (фініш)
3. Для кожної річки береться наступна випадкова крайова точка
4. Сервіс `IRiverPathfinder` будує шлях через A* з урахуванням висот
5. Клітинки вздовж шляху позначаються як річка в ObjectMap

> **Зверніть увагу:** Річка залежить від зовнішнього сервісу, який реєструється через `GraphBasedMapDataGenerator` та Zenject.
