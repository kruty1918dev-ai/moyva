# Height Source

← [Генератори](../nodes-generators.md) · [Graph System](../README.md)

**Категорія:** Generators · **Файл:** `HeightSourceNode.cs`

Головне джерело даних для будь-якого графа. Генерує карту висот (`float[,]`) через алгоритм Perlin noise.

---

## Порти

| Напрямок | Назва | Тип | Опис |
|---|---|---|---|
| **Output** 🟢 | HeightMap | `float[,]` | Згенерована карта висот (0.0–1.0) |

## Параметри

| Параметр | Тип | Опис |
|---|---|---|
| **Noise Settings** | `DataNoiseSettings` (SO) | ScriptableObject з налаштуваннями шуму |

## Як налаштувати `DataNoiseSettings`

| Поле | Що робить | Типове значення |
|---|---|---|
| `Scale` | Масштаб шуму. Більше значення → більші, плавніші пагорби | 50 |
| `Octaves` | Кількість "шарів" деталізації. Більше → детальніший рельєф | 4 |
| `Persistance` | Вплив кожної наступної октави (0–1). Вище → більше дрібних деталей | 0.5 |
| `Lacunarity` | Множник частоти між октавами. Вище → більше дрібних варіацій | 2.0 |
| `Seed` | Зерно генерації. Однаковий seed → однакова карта | 0 |
| `Offset` | Зміщення карти по X/Y | (0, 0) |
