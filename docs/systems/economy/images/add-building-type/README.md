# Add Building Type Screenshots

Покладіть у цю папку скріншоти для секції "Developer Guide: Як додати новий тип будівлі".

Рекомендовані імена файлів:
- step-1-open-registry-hub.png
- step-2-open-buildings-tab.png
- step-3-fill-form.png
- step-4-select-civilian-type.png
- step-5-created-row.png
- step-6-verify-in-game.png

Поради:
- Формат PNG, ширина 1600-2200px.
- Для читабельності збільшуйте UI Scale в Unity Editor до 125-150%.
- Якщо виділяєте важливі поля, використовуйте один стиль стрілок/рамок по всіх скрінах.
