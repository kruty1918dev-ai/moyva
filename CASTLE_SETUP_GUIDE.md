# Інструкція з налаштування системи детальної інформації про замок

## Готові компоненти

Система повністю реалізована на código рівні. Потребує налаштування тільки UI префаба.

## Кроки для подіння в дію

### 1. Підготовка UI префаба

У вашій сцені (або как prefab) створіть наступну структуру:

```
Canvas
└── CastleDetailedPanel (Panel component)
    ├── CastleNameText (TextMeshProUGUI)
    │   └── (Text: "Назва замку")
    ├── StatisticsText (TextMeshProUGUI)
    │   └── (Text: "Статистика")
    ├── ScrollView
    │   └── BuildingsListContainer (VerticalLayoutGroup)
    │       └── (кнопки додаються динамічно)
    └── CloseButton (Button)
```

### 2. Призначити префаб у WorldUIConfigSO

1. Знайдіть або створіть `WorldUIConfig` асет (Moyva/UI/World UI Config)
2. Перетягніть ваш `CastleDetailedPanel` префаб у поле **Castle Detailed Panel Prefab**
3. Збережіть

### 3. Перевірити структуру

Переконайтеся що коряга обраних компонентів точно збігаються:
- `CastleNameText` - для назви замку
- `StatisticsText` - для статистики
- `BuildingsListContainer` - контейнер для списку
- `CloseButton` - кнопка закриття

### 4. Налаштування стилів (опційно)

В методі `CreateBuildingItemButton()` у `CastleDetailedInfoPanelController.cs` можна змінити:

```csharp
// Розмір кнопки
rectTransform.sizeDelta = new Vector2(300, 40);

// Колір фону
image.color = new Color(0.2f, 0.2f, 0.3f, 0.8f);

// Розмір шрифту
textComponent.fontSize = 28;

// Висота елемента в списку
layoutElement.preferredHeight = 40;
```

## Архітектурні деталі

### Компоненти системи

| Компонент | Файл | Функція |
|-----------|------|---------|
| **Сигнал** | `OnBuildingInfoSignals.cs` | `CameraFocusBuildingSignal` |
| **Утиліта** | `CastleSettlementStatistics.cs` | Обчислення статистики |
| **Presenter** | `BuildingWorldInfoPresenter.cs` | Формування інформації |
| **Service** | `CameraFocusService.cs` | Переміщення камери |
| **Controller** | `CastleDetailedInfoPanelController.cs` | UI та кнопки |
| **Installer** | `ConstructionInstaller.cs` | Реєстрація компонентів |

### Потік виконання

```
1. Клік на замок
   ↓
2. BuildingWorldInfoPresenter обробляє => генерує детальну інформацію
   ↓
3. WorldInfoPanelRequestedSignal з контентом
   ↓
4. CastleDetailedInfoPanelController парсить список будівель
   ↓
5. Створює кнопки для кожної будівлі
   ↓
6. Гравець клікає на кнопку будівлі
   ↓
7. CameraFocusBuildingSignal відправляється
   ↓
8. CameraFocusService => закриває панель + переміщує камеру
```

## Розширення

### Додати іконки для різних типів будівель

Змініть в методі `FormatCastleInfoDetailed()`:

```csharp
var icon = info.IsWarehouse ? "🏪" : 
           info.IsBarn ? "🌾" : 
           info.IsHouse ? "🏠" : 
           "🏢";
```

### Додати сортування будівель

В методі `PopulateBuildingsList()` додайте сортування перед створенням кнопок:

```csharp
// Сортування по типу
var sorted = buildings
    .OrderBy(b => b.IsWarehouse ? 0 : b.IsBarn ? 1 : 2)
    .ToList();
```

## Часті проблеми

### Панель не з'являється
- Перевірте що `CastleDetailedPanelPrefab` призначено у `WorldUIConfigSO`
- Перевірте що структура префаба збігається з очікуваною

### Кнопки не створюються
- Перевірте що контейнер `BuildingsListContainer` існує і видимий
- Перевірте логи для помилок у парсуванні

### Камера не переміщується
- Перевірте що `CameraMovement` правильно інжектується
- Перевірте що позиція вищої координати коректна (x, y)

## Тестування

Система готова до запуску. Щоб протестувати:

1. Запустіть гру
2. Розмістіть замок на карті
3. Клікніть на замок
4. Перевірте що:
   - ✅ Показується інформація про замок
   - ✅ Видно список будівель
   - ✅ Кнопки клікабельні
   - ✅ Камера переміщується при кліку
   - ✅ Панель закривається при клікові

## Контакт та підтримка

При виникненні проблем перевірте:
1. Console для помилок
2. Структуру UI префаба
3. Назви компонентів (чутливі до регістру)
