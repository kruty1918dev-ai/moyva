# Превью Префабів у 2D режимі

## 🎯 Проблема

При перегляді префабів у Project вкладці, Unity за замовчуванням генерує превью в 3D проекції замість 2D. Це відбувається, оскільки вбудована система превью використовує perspektive камеру.

## ⚠️ Обмеження Unity

Unity's built-in preview system у Project вкладці має обмежену можливість налаштування. Превью камера автоматично встановлюється на `orthographic = false` для підтримки 3D раціоналізації.

## ✅ Вирішення

### Крок 1: Використовуйте Prefab Mode (найкращий варіант)

Двічі натисніть на префаб у Project вкладці → автоматично відкриється **Prefab Mode**:
- ✅ Scene View автоматично переключається на ортогональну проекцію
- ✅ Спрайти відображаються коректно (2D)
- ✅ Легше редагувати префаб у 2D

**Скорочак**: Двічі клікніть на префаб!

### Крок 2: Налаштуйте гізмо та превью

Використовуйте меню `Moyva → Windows → Prefab 2D Preview Setup`:

#### Функції:

1. **Фіксити встановлення 2D превью на всі префаби**
   - Налаштовує всі префаби в проекті
   - Встановлює правильні SpriteRenderer шейдери
   - Коригує Z-позицію на 0
   - Переконується, що масштаб = (1, 1, 1)

2. **Налаштувати гізмо для 2D**
   - Включає SpriteRenderer гізмо
   - Вимикає 3D гізмо (MeshRenderer, Colliders)

3. **Показати Sprite Outline Gizmo**
   - Показує контури спрайтів у Scene View

### Крок 3: Вручну для певного префаба

Меню `Moyva → Setup → Force 2D on Selected Prefab`:
- Виберіть префаб в Project
- Запустіть команду
- Префаб буде налаштований на 2D

## 🔧 Що робить налаштування

### Для кожного префаба:
- ✅ Встановлює `sortingOrder = 0` для SpriteRenderer
- ✅ Замінює неправильні шейдери на `Sprites/Default`
- ✅ Встановлює Z-позицію на `0`
- ✅ Переконується, що масштаб дорівнює `(1, 1, 1)`
- ✅ Вимикає MeshRenderer компоненти
- ✅ Коригує матеріали для спрайтів

### Для гізмо:
- ✅ Включає `SpriteRenderer` гізмо для видимості контурів
- ✅ Вимикає 3D гізмо (менше візуального шуму)
- ✅ Оптимізує Scene View для 2D роботи

## 🎨 Результат

### У Prefab Mode (РЕКОМЕНДУЄТЬСЯ):
- Scene View автоматично orthographic
- Perfect 2D превью
- Легко редагувати

### У Project вкладці (обмежено):
- Превью будуть мати менше 3D аспектів
- Спрайти будуть видны "плоско"
- Контури буді помітні з гізмо

## 📝 Editor Scripts

- `Prefab2DPreviewSetup.cs` 
  - Ініціалізація та налаштування сцен превью
  - Команди меню для конфігурації
  
- `Gizmo2DSetup.cs` 
  - Налаштування гізмо
  - Постпроцесування префабів
  
- `PrefabPreview2DWindow.cs` 
  - Editor Window для управління налаштуванням
  - Пакетне налаштування префабів

## 🚀 Використання

### Програмно:
```csharp
// Налаштувати префаб для 2D
var prefab = PrefabUtility.LoadPrefabContents("Assets/Moyva/Prefabs/Units/Unit-warrior-01.prefab");
// Налаштування будуть застосовані автоматично
PrefabUtility.SaveAsPrefabAsset(prefab, "Assets/Moyva/Prefabs/Units/Unit-warrior-01.prefab");
```

### Через UI:
1. `Moyva` → `Windows` → `Prefab 2D Preview Setup`
2. Виберіть потрібну операцію
3. Натисніть кнопку

## 🎯 Найкращий워크флоу

1. **Редагування**: Dvicлік на префаб → Prefab Mode автоматично 2D ✨
2. **Перегляд**: Подивіться на спрайти у Prefab Mode
3. **Налаштування**: Якщо потрібно, запустіть смену гізмо

## ⚡ Скорочаки Scene View

| Action | Effect |
|--------|--------|
| `2` на Numpad | Переключ на 2D (Orthographic) |
| `0` на Numpad | Переключ на 3D (Perspective) |
| `Shift + F` | Focus на вибрану об'єкт |
| `Right Click + Drag` | Панорамування камери |

## ✅ Перевірка

Команда меню **Moyva → Setup → Force 2D on Selected Prefab** поверне:
- ✅ "Префаб налаштовано на 2D режим!" - успішно
- ❌ "Вибраний об'єкт не є префабом!" - виберіть префаб

## 🔍 Подальші поліпшення

Якщо превью все ще виглядає 3D:

1. **Переконайтесь, що SpriteRenderer налаштовано**:
   - Запустіть `Moyva → Windows → Prefab 2D Preview Setup`
   - Натисніть `Фіксити встановлення 2D превью на всі префаби`

2. **Вимикіть 3D гізмо**:
   - Запустіть `Moyva → Gizmo → Hide 3D Bounds`

3. **Використовуйте Prefab Mode**:
   - Двічі клікніть на префаб (найкращий варіант!)
